1. 문제풀이 시 20~30분 고민했는데 알고리즘이 안떠오르면 답
2. 답 코드를 보고 이해한다. 혼자서 스스로 다시 짜본다. 체크해놓고 나중에 반드시 다시 풀어야한다.
3. 프로그래머스 사이트 -> 코딩 테스트 연습 -> 모든 문제 -> 레벨 1,2 에 있는 문제를 가능하면 다 풀자

4. 이제 완탐을 푼다
    1. BFS 혹은 DFS
    2. 자유롭게 다른 유형
